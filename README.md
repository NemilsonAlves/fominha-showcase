# Fominha SaaS — Automação de Pedidos e Delivery no WhatsApp

Este repositório contém a documentação da arquitetura e demonstração visual pública da plataforma **Fominha**.

## 📊 Visão Geral
O Fominha é um SaaS de automação de pedidos para restaurantes e lanchonetes integrado ao WhatsApp. Ele permite que o cliente interaja com um cardápio digital dinâmico diretamente pelo WhatsApp, escolha produtos, finalize o pedido e pague de forma instantânea via PIX com geração automática de QRCode e notificação do status do preparo em tempo real.

-   **Linguagens/Stack:** Node.js, Express, Supabase, Webhooks.
-   **Integração de Pagamento:** InfinityPay API v2 (Pix).
